🔄 Cara Update Aplikasi

Ikuti langkah-langkah berikut agar aplikasi selalu versi terbaru

1
Tutup Aplikasi YussXy
Geser aplikasi ke atas dari recent apps untuk menutup paksa aplikasi YussXy.
2
Info Aplikasi YussXy
Tahan ikon YussXy → pilih Info Aplikasi (i).
3
Hapus Cache Aplikasi
Pilih Storage/Penyimpanan → klik Hapus Cache.
4
Hapus Cache Chrome
Tahan ikon Chrome → Info Aplikasi → Storage → Hapus Cache.
5
Buka Tab Chrome
Buka Chrome → klik ikon kotak (Tab Switcher).
6
Hapus Semua Tab
Klik 3 titik → pilih Hapus semua tab.
✓
Selesai!
Buka kembali aplikasi YussXy. Aplikasi sekarang akan terupdate ke versi terbaru! 🎉
