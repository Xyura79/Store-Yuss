🔄 Cara Update Aplikasi

Ikuti langkah-langkah berikut

1
Tutup Aplikasi
Geser aplikasi ke atas dari recent apps untuk menutup paksa.
2
Info Aplikasi
Tahan ikon → pilih Info Aplikasi (i).
3
Hapus Cache
Pilih Storage → klik Hapus Cache.
4
Cache Chrome
Tahan Chrome → Info Aplikasi → Storage → Hapus Cache.
5
Tab Chrome
Buka Chrome → klik ikon kotak (Tab Switcher).
6
Hapus Tab
Klik 3 titik → pilih Hapus semua tab.
✓
Selesai
Buka kembali aplikasi. Sekarang sudah terupdate ke versi terbaru!
